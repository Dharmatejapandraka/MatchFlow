# 🩸 MatchFlow

MatchFlow is a Smart Blood Donation Platform built using Flask, HTML, CSS, JavaScript, and SQLite.

## 🚀 Features

- User Registration & Login
- Blood Request System
- Donor Matching
- Notifications
- Accept Blood Requests
- Donation Completion Tracking
- Admin Dashboard
- Responsive Mobile-Friendly UI
- Active / Inactive Donor Status
- Donation History Tracking

## 🛠️ Technologies Used

- Python Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## 📱 Responsive Design

Works on:
- Desktop
- Mobile Phones
- Tablets
